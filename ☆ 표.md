## 표

+ ### 테두리
    + 통합
      ```angular2html
      border-collapse : collpase
      ```
    + 간격
      ```angular2html
      border-spacing : 크기
      ```      

+ ### 너비&높이
    ```angular2html
    width/height: 크기
    ```
+ ### 너비 고정
    ```angular2html
    table-layout: fixed
    ```
+ ### 정렬
    + 수평
      ```
      text-align: reft/right/center
      ```
    + 수직
      ```angular2html
      vertical-aline: top/bottom
      ```

+ ### 생성(display)
  + 블록요소 -> 테이블 
     ```
     display : table
     ```
  + 블록요소 -> 행
     ```
     display : table-row
     ```
  + 블록요소 -> 셀
     ```
     display : table-cell
     ```
